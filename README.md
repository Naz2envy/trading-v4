# TradeSea Complete

Final manual trading journal with Supabase authentication, cloud sync, screenshots, accounts and funded-risk tracking, daily reviews, responsive custom dashboard widgets, interactive equity curve, dark mode and PWA support.

## Setup
1. Run `supabase-setup.sql` in Supabase SQL Editor.
2. Add `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY` to Vercel.
3. Upload all files in this folder to the repository root.
4. Deploy using Node 22.x.
